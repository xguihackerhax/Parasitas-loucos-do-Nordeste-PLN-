# Parasitas-loucos-do-Nordeste-PLN-
Parasitas Loucos do Nordeste é um jogo top-down de sobrevivência contra hordas de parasitas, com múltiplas armas, bosses, sistema de torretas, escudos, paralisadores, trilha sonora dinâmica, efeitos sonoros gerados por código e menu secreto de hacks. Desenvolvido 100% em HTML, CSS e JavaScript.
